Usage
=====
The usage of modules available in ``PyAlgo``

Importing
---------
``PyAlgo`` module can be imported in the following ways:

```python
import pyalgo
import pyalgo.{module_name}
from pyalgo import *
from pyalgo.{module_name} import *
```
